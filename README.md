
# Data Prototype

A learning and prototype repository for data science and applied AI projects.

---

## Week 1 — API → CSV (Open-Meteo)

Fetches hourly weather data for Ljubljana via the [Open-Meteo API](https://open-meteo.com),  
exports it as a CSV, and visualizes basic results.

Notebook: `notebooks/01_api_to_csv.ipynb`  
[![Open In Colab](https://colab.research.google.com/assets/colab-badge.svg)](
https://colab.research.google.com/github/mihahafner/data-prototype/blob/main/notebooks/01_api_to_csv.ipynb)
